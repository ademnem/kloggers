# Learning How to Use Winsock2


In order to compile `main.cpp` use:
```{bash}
g++ main.cpp -o main
./main
```


## APIs
The `Winsock` or `winsock.h`API was used in order to communicate using Windows sockets.

Starter [documentation](https://learn.microsoft.com/en-us/windows/win32/winsock/getting-started-with-winsock) for Winsock was used.
